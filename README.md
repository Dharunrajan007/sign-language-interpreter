# Sign Language Recognition & Hospital Kiosk

This project provides a robust, real-time hand gesture recognition system designed for emergency medical communication. It leverages advanced landmark detection to recognize a custom set of sign language gestures, allowing patients who are unable to speak to effectively communicate their needs to hospital staff.

## Features
- **Real-Time Recognition:** Detects gestures instantly using a webcam feed.
- **Custom Gestures:** Supports medical-specific gestures (e.g., pain, water, call doctor).
- **Web Interface:** Includes a modern, reactive web UI for interacting with the gesture detection system.
- **Fast Inference:** Optimized architecture for minimal latency.

## Setup Instructions
1. Install requirements: `pip install -r requirements.txt`
2. Run the application: `python app.py` (or `python server.py` depending on the entry point)
3. Access the provided localhost URL in your browser.
